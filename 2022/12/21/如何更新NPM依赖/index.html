<h4 id="npm-outdated"><a href="#npm-outdated" class="headerlink" title="npm outdated"></a>npm outdated</h4><p>npm 内置 该命令将检查每个已安装的依赖关系，并将当前版本与 npm registry 中的最新版本进行比较。它在终端打印出一个表格，概述了可用的版本。</p> <ul> <li>Current：是当前安装的版本。</li> <li>Wanted：是根据 semver 范围内的软件包的最大版本。</li> <li>Latest：是在 npm registry 中被标记为最新的软件包版本。</li> </ul> <p>npm update 命令会更新到 wanted 版本</p> <h4 id="npm-check-updates"><a <li>Red （显示红色） &#x3D; major （主版本，或者说是大版本）</li> <li>Cyan （显示青色） &#x3D; minor（次要版本）</li> <li>Green（显示绿色） &#x3D; patch （补丁版本）</li> </ul> <p>ncu –target [patch, minor, latest, newest, greatest]
