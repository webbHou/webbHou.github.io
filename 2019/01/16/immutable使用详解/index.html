<h3 id="什么是immutable-Data"><a href="#什么是immutable-Data" class="headerlink" title="什么是immutable Data"></a>什么是immutable Data</h3><p>immutable Data就是一旦创建就不可变的数据，每次的修改和添加删除操作都会返回新的immutable Data数据，Immutable 实现的原理是 Persistent Data Structure（持久化数据结构），immutable会共享相同的数据结构，当两个对象的hashCode相同，值就相同。当修改时只会对有影响的节点进行修改，其余的继续共享，避免了deepclone把所以节点都复制一遍带来的性能损耗，immutable解决了复杂类型引用赋值带来的数据隐患。</p> <h3 id="immutable的优点"><a href="#immutable的优点" class="headerlink" title="immutable的优点"></a>immutable的优点</h3><ul> <li>降低了复杂可变数据带来的复杂度</li> <li>节省内存(共享相同的节点)</li> <li>函数式编程</li> </ul> <h3 id="常用的immutable数据结构"><a href="#常用的immutable数据结构" class="headerlink" title="常用的immutable数据结构"></a>常用的immutable数据结构</h3><ul> <li>Map：键值对集合，对应于 Object，ES6 也有专门的 Map 对象</li> <li>List：有序可重复的列表，对应于 Array</li> <li>Set：无序且不可重复的列表</li> </ul> <h3 id="常见用法"><a href="#常见用法" class="headerlink" title="常见用法"></a>常见用法</h3><ul> <li>immutableA.immutableB(bb) A融合B</li> <li>Immutable.Map({1:2}) 创建一个Map</li> <li>Immutable.is(比较两个immutable对象的值 &#x3D;&#x3D;&#x3D;比较内存)</li> <li>Immutable.fromJS([1,2]) 生成不可变对象(支持数据嵌套)</li> <li>Immutable.List([1,2]}) 生成不可变对象(不支持数据嵌套)</li> <li>immutableData.toJS() 不可变数据生成 JavaScript 对象</li> <li>immutableA is immutableB 判断两个数据引用是否一致</li> <li>Immutable.toObject 转化为对象</li> <li>Immutable.toArray 转化为数组</li> </ul> <h3 id="List常用方法"><a href="#List常用方法" class="headerlink" title="List常用方法"></a>List常用方法</h3><ul> <li>Immutable.List.of([1,3]) 创建一个List</li> <li>Immutable.List([1,2]) 生成不可变List 数据</li> <li>immutableA.size&#x2F;immutableA.count() 查看List 大小</li> <li>Immutable.List.isList(x) 判断是不是List</li> <li>immutableData.get(0)&#x2F;immutableData.set(0,el) 根据索引读写对象</li> <li>immutableData.getIn [1, 2]&#x2F;immutableData.setIn [1, 2] 读写嵌套数组中的数据</li> <li>immutableB.update(1,(x) -&gt; x + 1)&#x2F;immutableB.updateIn [2, 1], (x) -&gt; x + 1 更新不可变数据 接受函数</li> <li>immutableData.find (x)&#x3D;&gt;x&gt;1 查找</li> <li>immutableData.filter (x) -&gt;x&gt;1 过滤</li> <li>immutableData.filterNot (x) -&gt; x &lt;&#x3D; 1 取反过滤</li> <li>immutableData.sort (a, b) -&gt; return a&gt;b?-1:1 排序</li> <li>immutableData.forEach (a, b)&#x3D;&gt; 遍历</li> </ul> <h3 id="Map常用方法"><a href="#Map常用方法" class="headerlink" title="Map常用方法"></a>Map常用方法</h3><ul> <li>Immutable.Map.isMap(x) 判断是不是map</li> <li>immutableData.get(‘a’)&#x2F;immutableData.getIn([‘a’,’b’]) 直接&#x2F;嵌套更新获取</li> <li>immutableA.set(‘a’, 1)&#x2F;immutableA.setIn( [‘a’, ‘b’], 1) 直接&#x2F;嵌套更新设置</li> <li>immutableA.update (‘a’, (x) -&gt; x + 1)&#x2F;immutableA.updateIn ([‘a’, ‘b’], (x) -&gt; x + 1) 直接&#x2F;嵌套更新数据</li> <li>immutableA.merge(immutableC) 合并对象</li> <li>immutableData.has(‘key’) 判断是否存在该属性</li> <li>immutableData.filter (value, key) -&gt; value is 1 属性过滤</li>
