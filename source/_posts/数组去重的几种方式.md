### new Set 简单直接


接下来就是两个需要遍历的方法了
### Array.splice
```bash
  for(i = 0; i < arr.length; i++){
    for(j = i + 1; j < arr.length; j++){
      if(arr[i] == arr[j]){
        arr.splice(j,1);
        j--;
      }
    }
  }
```

### Array.push
```bash
  function uniqueArray(arr){
      var retArray = [];
      for(var i = 0; i < arr.length; i++){
          if(retArray.indexOf(arr[i]) < 0){
              retArray.push(arr[i]);
          }
      }
      return retArray;
  }
```
